🧚‍♂️ Wiki Legend of Neverland

Una enciclopedia interactiva y visual dedicada al mundo mágico de Neverland. Este proyecto presenta de manera organizada la información sobre las hadas florales, las clases de personajes y el vasto mundo del juego, utilizando una estética vibrante y moderna.

🌟 Características Principales

Interfaz Dinámica: La web cuenta con un fondo animado mediante gradientes CSS que cambian suavemente a lo largo del tiempo.
Diseño Glassmorphism: Implementación de efectos de "cristal esmerilado" en la barra de navegación y los encabezados utilizando backdrop-filter: blur.
Componentes Interactivos:
Carrusel de Imágenes: Galería visual para mostrar el mundo de Neverland.
Tarjetas de Hadas: Clasificación por elementos (Fuego, Agua, Aire, Tierra y Sagrada) con gradientes temáticos personalizados.
Sección de Clases: Información detallada sobre las especializaciones de combate (Ninja, Lancero, Explorador, Guerrero, Artesano y Mago).
Navegación Fluida: Barra de navegación pegajosa (sticky) con enlaces internos a las secciones principales.

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica del contenido.
CSS3 (Custom Styles):
Variables CSS para gestión de colores y gradientes.
Animaciones keyframes para efectos de entrada (fade-in) y fondos móviles.
Diseño adaptable mediante Flexbox y Grid.
Bootstrap 5.3.8: Utilizado para la rejilla (grid), el carrusel, la navegación responsiva y el estilo de las alertas informativas.
Google Fonts: Tipografías Nunito y Poppins para una lectura clara y moderna.

🆚 Clases de Personajes

Detalle de las mecánicas de juego y especializaciones:
Ninja: Daño cuerpo a cuerpo (Búho Ardiente/Sombrío).
Lancero: Poder dracónico (Caballero/Luchador Dragón).
Explorador: Combate a distancia (Tirador/Cazador de viento).
Guerrero: Fuerza bruta (Gladiador/Berserker).
Artesano: Creaciones mecánicas (Armero/Mecánico).
Mago: Magia y sanación (Juez/Sacerdote).

🚀 Instalación y Uso

Asegúrate de mantener la estructura de carpetas para que las imágenes y los estilos carguen correctamente.
Abre el archivo neverland.html en cualquier navegador moderno.
La navegación requiere conexión a internet para cargar las librerías de Bootstrap y las fuentes de Google Icons.
